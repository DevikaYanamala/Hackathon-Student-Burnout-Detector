# ClearHead — Student Burnout Detector

An interactive, premium client-side web application designed to help students analyze daily habits, detect burnout risk levels, and receive personalized wellbeing recommendations. This project was developed as a hackathon solution.

## 🚀 Live Demo
Simply open the `index.html` file in any modern web browser to run the app instantly. No installations, server setups, or external APIs are required.

## ✨ Key Features
- **Burnout Assessment:** A quick, 6-question diagnostic covering sleep, study load, screen time, exercise, stress, and breaks.
- **Dynamic Risk Score:** Real-time risk classification (Low, Moderate, High) visualized through an interactive Chart.js donut gauge.
- **Daily Check-In Tracker:** Log 7 key daily wellness habits with an automated focus score calculator.
- **Goal Setter & Tracker:** Set daily study goals and track completion percentages.
- **Distraction Logging:** Identify and log daily distractions (e.g., social media, gaming) to map time-wasting habits over time.
- **Progress Dashboard:** Tracks daily habit consistency (last 7 days) and score trends (last 14 days) using responsive line charts.
- **Evergreen Design:** Features a futuristic space-themed dark mode using CSS glassmorphism, glowing accents, and drift animations.

## 🛠️ Technology Stack
- **Frontend:** Semantic HTML5, Vanilla CSS3 (Custom variables, glassmorphism, responsive grid layouts)
- **Logic:** Native ES6+ JavaScript (Clean state management, LocalStorage integration for offline data persistence)
- **Charts:** [Chart.js](https://www.chartjs.org/) (CDN version 4.4.1 for lightweight rendering)
- **Typography & Icons:** DM Sans & DM Serif Display (via Google Fonts), native SVGs, and Emojis

## 🔒 Privacy & Offline First
All data logged (check-ins, scores, habits, and distractions) is saved securely and locally in your browser's `localStorage`. No data is uploaded to external servers, ensuring 100% user privacy.
